## Features

<CardGrid stagger>
  <Card title="Rich Expression Language" icon="document">
    Powerful expressions with arithmetic, logical, comparison, and collection
    operations. Support for quantifiers, pattern matching, and conditional
    expressions.
    <br />
    <a href="/reference">Learn more</a>
  </Card>
  <Card title="Composable Policies" icon="setting">
    Build complex policies from simpler policies by exporting decisions that can
    be used by other policies.
    <br />
    <a href="/reference/exporting-and-importing-rules">Learn more</a>
  </Card>
  <Card title="JavaScript Integration" icon="setting">
    Seamlessly integrate TypeScript modules for complex logic. Built-in modules
    for data manipulation, cryptography, and more.
    <br />
    <a href="/reference/using-typescript">Learn more</a>
  </Card>
  <Card title="Run as a Service" icon="setting">
    Run Sentrie as a service to evaluate policies using a well-defined and
    minimal HTTP API.
    <br />
    <a href="/running-sentrie/serving-policies">Learn more</a>
  </Card>
  <Card title="CLI Reference" icon="open-book">
    The Sentrie CLI provides a powerful set of tools for managing your policies
    and JavaScript modules.
    <br />
    <a href="/cli-reference">Learn more</a>
  </Card>
  <Card title="Type-Safe Inputs" icon="pencil">
    Comprehensive type system with shapes, constraints, and built-in
    validations.
    <br />
    <a href="/reference/types-and-values">Learn more</a>
  </Card>
</CardGrid>
